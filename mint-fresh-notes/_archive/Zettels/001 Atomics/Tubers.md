# Tubers
```ad-example

- Potatoes
- Sweet Potatoes
- Jerusalem Artichokes

```
## Selection
When selecting tubers:
- Firm
- Smooth
- Even size
- No greening
- Free from shoots
## Preperation
- Wash
- Peel (or not)
## Storage
- Cool, dark place
