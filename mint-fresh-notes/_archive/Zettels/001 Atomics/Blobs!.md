# Blobs!
## Idea Types
- **Simple Action**: S + V...
	- *I + throw...*
- **Medium Action**: S + V + O
	- *I + throw + rock*
- **Complex Action**: S + V + O<sub>1</sub> + P + O<sub>2</sub>
	- *I + throw + rock + at + Player A*
- **Memories**: Cause Phrase -> Effect Phrase (+ or / or -)
	- **Cause Phrase**: Action Phrases
	- **Effect Phrase**: S + V?
