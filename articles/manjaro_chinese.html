getting chinese layout on manjaro

For some god forsaken reason, the otherwise well documented Linux distributions lack instructions for setting up keyboard layouts for other languages. I learn a bit of Chinese, so here is how to get a Chinese keyboard layout on Manjaro Linux through fcitx.

Fcitx is an input tool that lets you use other layouts. It is not intuitive.

  1. Install the necessary packages: sudo pamac install fcitx fcitx-googlepinyin fcitx-ui-light kcm-fcitx fcitx-configtool
  2. Open fcitx-configtool and add ‘Google Pinyin’ to the input methods.
  3. Reboot (or log out)
