even more hydrocarbons

Name Group Suffix
Alcohol -OH -ol
Aldehyde =O (at the end) -al
Ketone =O (middle) -one
Carboxylic acid (alcohol + aldehyde) -oic acid
Ester O=C-O-C A-yl-B-anoate
Amine -NH3 -amine

When you name an ester, make sure it’s the alcohol part first, carboxylic acid part section. i.e. It is 1-butylbutanoate.

Carboxylic acid + alcohol → ester + water

If C atom attached to OH group is attached to only one other C atom, it is a primary alcohol. If C atom attached to OH group is attached to only one other C atom, it is a primary alcohol.
